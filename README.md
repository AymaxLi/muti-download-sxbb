# Muti-Download-SXBB
针对 [师兄帮帮网](https://sxbb.me) 的资源批量下载爬虫脚本

# Usage
install the dependencies
``` bash
npm install
```

set the target website and the result dir
``` javascript
let url_page = 'https://sxbb.me/WOuba' // 目标页面
let resultDir = `${__dirname}/result` // 文件保存路径
```

then just run it
``` bash
npm start
```
